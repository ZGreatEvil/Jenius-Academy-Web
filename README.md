## About

Jenius Academy Web is a multi-page static website designed and developed as an academic project. The site is structured around separate HTML pages, a shared CSS stylesheet, JavaScript for interactivity, and supporting assets.

## Getting Started

No build step or dependencies required. Open any HTML file directly in a browser.

```bash
git clone https://github.com/ZGreatEvil/Jenius-Academy-Web.git
cd Jenius-Academy-Web
```

Then open the entry page from the `html/` folder in your browser, or use a local server:

```bash
# Using Python
python -m http.server 8000
```

Then visit `http://localhost:8000`.
